3. 해석

600명은 여유 범위 안에 있습니다. 성공률 99.94%, 정상 응답 263ms 이하, 네트워크는 회선의 약 10%만 사용. 600명 이상으로도 상당한 여유가 있습니다.
  • 정상 사용 경험은 빠름: 첫 페이지 로딩 후 탐색은 p95 약 263ms.
  • 최초 접속(콜드)이 가장 무거운 순간: 첫 페이지가 이미지·오디오(~15MB)를 받아 p95 약 20초. 사용자당 1회성 비용이며, 자산 경량화(적용 완료)와 입장 분산으로 완화됩니다.
  • 네트워크는 병목이 아님: 피크 약 104.1 Mbps로 1 Gbps의 일부에 불과.
  • AI 로테이션 안전: 운영에 적용된 AI Studio 10개 키 풀이 생성 부하를 별도로 처리. 본 테스트는 웹 서빙 계층을 검증.

4. 목요일(600명) 권장 사항

  • 입장 분산: 가능하면 1~2분에 걸쳐 입장하도록 유도하여 동시 콜드 로딩 부담 완화.
  • 모니터링 대기: 수업 시작 시각에 컨테이너 상태/오류 로그 실시간 확인.
  • 용량 측면 추가 조치 불필요 — 600명은 여유 있게 처리됨.
